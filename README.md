### 基本功能
- 谷歌 Material 设计
- 响应式设计
- 内置3套主题，供自由切换
- 多种颜色方案，有18个预定义的方案
- Flex Layout 强大的布局设置
- 1500+ 图标, 包括 900+ 官方 material design 图标
- Sass/Scss CSS, 很容易通过改变一些变量的更新
- 模块代码，添加和删除非常轻松
- Webpack 2 构建项目
- 通过 NPM 管理包
- 跨浏览器支持
- 免费更新

### 使用入门

```
npm i -g @angular/cli
git clone https://github.com/stbui/angular2-material2-bootstrap4-admin-app.git
```


进入项目目录
```
npm install
npm run start
```
打开你的浏览器，访问
```
http://127.0.0.1:4200
```

### 发布

```
ng build --prod --aot --env=prod --extract-css
```
